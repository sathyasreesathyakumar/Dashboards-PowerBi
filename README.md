**Athletics Championship Performance Analysis Dashboard**



**Project Description**



This project is an interactive Power BI dashboard developed to analyze and compare athlete performance in multiple track and field events from the Asian Athletics Championship. The dashboard focuses on comparing men's and women's performances across different events, including 200m, 400m Hurdles, and 800m races.



The objective of this project is to provide an intuitive and user-friendly analytical platform that enables users to explore athlete records, reaction times, rankings, and event outcomes through dynamic visualizations and navigation.



**Key Features**



**Interactive Navigation System**



\* Designed a Home Page with navigation buttons.

\* Implemented page navigation functionality using Power BI action buttons.

\* Users can seamlessly switch between different event pages such as:



&#x20; \* 200m

&#x20; \* 400m Hurdles

&#x20; \* 800m



**Performance Comparison**



\* Compared men's and women's athletic performances across multiple events.

\* Analyzed race results, rankings, and reaction times.
